# 🏠 House Price Prediction

This is a beginner Machine Learning project that predicts the price of a house using **Linear Regression**.

## 📌 Features

- Predict house price
- Data preprocessing
- Handle missing values
- One Hot Encoding
- Linear Regression model
- Model evaluation
- User input prediction
- Save trained model using Joblib
## 📌 Features
## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib

## 📊 Dataset Features

- Area (SqFt)
- Number of Rooms
- Build Year
- Location
- Street Type
- Furnishing
- Property Type
- Swimming Pool Availability

**Target Variable**

- House Price
  
## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries.

```bash
pip install pandas numpy scikit-learn matplotlib seaborn joblib
```

3. Run the Python file.

```bash
python main.py
```

4. Enter the house details when prompted.
5. The model will predict the estimated house price.
